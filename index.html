Heathcare Management System

Instructions

  1. To create new patients, you must have all fields filled.
  2. To create new doctors, you must have all fields filled.
  3. To create new drugs, you must have all fields filled.
  4. To create new appointments, you must have all fields filled, and have both patients and doctors registered.
  5. To create new prescriptions, you must have all fields filled, and have patients, doctors, and drugs registered.
  6. To make adjustments/updates, you can click on detail button under details section, and then change accordingly. All fields must be filled as well.
  7. To delete records, you can click on the X button under remove section.
